# ⌨️ Typing Speed Test (Python)

A simple command-line based Typing Speed Test game built using Python.
This project helps users measure their typing speed, accuracy, and errors in real-time.

---

## 🚀 Features

* 📄 Loads random sentences from a file (`sentence.txt`)
* ⚡ Calculates typing speed in **Words Per Minute (WPM)**
* ❌ Counts typing errors (word-based comparison)
* 🎯 Calculates typing accuracy (%)
* 🔁 Continuous testing loop (play multiple times)

---

## 🛠️ Technologies Used

* Python 3
* Standard Libraries:

  * `time`
  * `random`

---


## 🧠 How It Works

1. Sentences are stored in `sentence.txt` (one per line)
2. Program randomly selects a sentence
3. User types the sentence
4. Program calculates:

   * ⏱️ Time taken
   * ⚡ Speed (WPM)
   * ❌ Errors
   * 🎯 Accuracy
